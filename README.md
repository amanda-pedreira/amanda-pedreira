## 🛒 E-commerce de Suplementos

Projeto acadêmico desenvolvido utilizando **Angular, TypeScript e Firebase** para simulação de uma plataforma de e-commerce.

A aplicação contempla:

- CRUD de produtos;
- CRUD de usuários;
- CRUD de funcionários;
- CRUD de contatos;
- Controle de estoque;
- Destaques de produtos;
- Carrinho de compras;
- Autenticação;
- Integração com banco de dados.

**Obs:** inicialmente o projeto utilizava uma API REST simulada através de arquivos JSON. Posteriormente foi migrado para Firebase para persistência dos dados e publicação da aplicação.

**Tecnologias utilizadas:**

Angular • TypeScript • Firebase • API REST
